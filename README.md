# Arief Labs — AI Chat Client

> A fast, private desktop app for Claude & OpenAI. Pay only for what you use.

---

## Why Arief Labs?

- **No subscription limits** — use Claude and OpenAI APIs directly
- **Real cost tracking** — see exactly what every conversation costs
- **Your data stays local** — conversations stored on your machine, never our servers
- **Actively maintained** — built and updated by a real developer

## Features

- 💬 **Claude & OpenAI** — switch providers with one click
- 💰 **Cost dashboard** — monthly spend, daily average, savings vs Pro plan
- 📎 **File uploads** — images, PDFs, code files, text files
- 🎨 **Blender MCP** — connect Claude to Blender via MCP protocol
- 🌙 **Dark mode** — easy on the eyes
- 📤 **Export conversations** — save as Markdown or plain text
- 📥 **Import conversations** — load previous exports back in
- 🔒 **Encrypted API keys** — stored using OS keychain, not plain text
- 📌 **Pin & tag conversations** — organise your chat history
- ⌨️ **Keyboard shortcuts** — Cmd+K command palette, F12 DevTools

---

## Getting Started

1. Download the installer for your platform above
2. Install and open Arief Labs
3. Get your API key:
   - **Claude:** [console.anthropic.com](https://console.anthropic.com) → API Keys
   - **OpenAI:** [platform.openai.com](https://platform.openai.com) → API Keys
4. Paste your key when prompted
5. Start chatting
